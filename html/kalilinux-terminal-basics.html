KaliLinuxことはじめ

Kali Linux でのターミナル起動とコマンド入力の基本

質問

Kali Linux をインストールしたものの、「コマンドを実行する」とはどこで何をすれば よいのかが分からない。ターミナル（端末）はどう起動して、どうやってコマンドを 打ち込み、結果を読み取ればいいのか。

原因（理由）

  • Kali Linux に同梱されているセキュリティツールの大半は コマンドラインで操作する 作りになっており、ターミナル操作は避けて通れない
  • Kali のデフォルトデスクトップ環境（Xfce）にはターミナルの起動経路が複数あり、最初は迷いやすい
  • 「コマンドを打って Enter で実行 → 結果が表示される → 次の入力待ちに戻る」という基本サイクルさえ掴めれば、以後の作業がぐっと楽になる

対策（手順）

1. ターミナルの起動方法

  • 方法 A（推奨）：画面上部のパネル（タスクバー）にある「黒い四角に >_」のアイコンをクリック
  • 方法 B：画面左上「アプリケーション」メニュー → Usual applications → System → Terminal Emulator を選ぶ
  • 方法 C：キーボードショートカット Ctrl+Alt+T（環境によっては割り当てられている）

2. プロンプトの読み方

ターミナルを起動すると、次のような行が表示されて入力待ちになる：

kali@kali:~$
  • 最初の kali = ログイン中のユーザー名
  • @ の後ろの kali = コンピューター（ホスト）の名前
  • :~ = 今いるフォルダ（ディレクトリ）。~ はホームディレクトリの省略記号
  • 末尾の $ = 一般ユーザーでの入力待ち（管理者なら # に変わる）

この $ の右側に、コマンドを打ち込んでいく。

3. コマンドの入力と実行

キーボードからコマンドを打ち、最後に Enter を押すと実行される。試しに次のコマンドを 1 行ずつ打ち、動きを確認してみる：

kali@kali:~$ pwd
/home/kali
kali@kali:~$ whoami
kali
kali@kali:~$ date
2026年  5月 26日 火曜日 12:00:00 JST
kali@kali:~$ ls
Desktop  Documents  Downloads  Music  Pictures  Public  Templates  Videos
  • pwd = 今いるディレクトリのパスを表示する（print working directory）
  • whoami = ログイン中のユーザー名を表示する
  • date = 現在の日時を表示する
  • ls = 今いるディレクトリの中身を一覧する（list）

コマンドを実行すると結果が下に出力され、続けて次の kali@kali:~$ が現れる。これが「次の入力を受け付けている」状態。

4. 管理者権限が必要なときは sudo を付ける

システム全体に影響する操作（パッケージのインストール、設定ファイルの編集 など）には、コマンドの頭に sudo を付ける。実行するとパスワードを聞かれる：

kali@kali:~$ sudo apt update
[sudo] password for kali:

ここで パスワードを打っても画面には何も表示されない（カーソルすら動かないように見える）。これは Linux の仕様であり、ちゃんと入力されている。打ち終わったら Enter。

5. 知っておくと楽になるキー操作

  • ↑（上矢印）：直前に実行したコマンドを呼び戻す（長いコマンドを打ち直さずに済む）
  • Tab：コマンド名やファイル名を途中まで打って Tab を押すと、続きを自動補完してくれる
  • Ctrl+C：実行中のコマンドを中断する
  • Ctrl+L（または clear コマンド）：画面をクリアして上にスクロール

6. ターミナルを閉じる

exit と入力して Enter、またはウィンドウ右上の「×」ボタンで閉じる。ターミナルを閉じても OS 自体は動き続ける。シャットダウンしたいときはデスクトップ右上の電源メニューから操作する。

注意点

  • コマンドのつづりは 大文字小文字を厳密に区別する。Ls や LS は ls とは別物として扱われる
  • スペースや記号にも意味がある。最初のうちは打ち間違いを避けるため、本記事や信頼できる資料から コピー＆ペースト するのが安全
  • Ctrl+C で止めても、ファイルへの書き込みなど すでに行われた操作は巻き戻らない
  • 「ターミナル」「端末」「シェル」はだいたい同じものを指す（厳密には別概念だが、初学のうちは同じものと捉えてよい）
